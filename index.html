Bootstrap5 Introduction

  • Bootstrap5 is the newset version of Bootstrap3 & 4, which is the most popular HTML, CSS and JavaScript framework, to create responsiveness and mobile-first websites, with new components, faster stylesheet and more responsiveness.
  • Bootstrap includes HTML and CSS based design templates for Typography, forms, buttons, lists, tables, navigation, carousels and many others as well as optional JavaScript plugins.
  • Bootstrap5 supports lastest, all major browsers and platforms. But Internet Explorer11 and down is not supported.
  • The major difference between Bootstrap5 and Bootstrap3&4 is switched to Vanila JavaScript instead of JQuery.

BS5 Grid system

  • Grid system is built with flexbox and it allows upto 12 columns across the page.
  • If we don't want to use all 12 columns individually, we can group them together to create wider columns
  • Grid system is a responsive, and the columns will re-arrange automatically depending on the screen size
